# ADR 0000: 클린룸 부트스트랩

## 상태
승인됨

## 배경
원본 구현 계획은 실거래 코드, 비밀 정보, Redis, 브로커 커넥터를 제외한 새 저장소를 요구한다.

## 결정
Buy-Low-Sell-High를 PostgreSQL 전용 클린룸 프로젝트로 부트스트랩하고, Python 엔진과 TypeScript 대시보드 스켈레톤을 둔다. 소스 트리에 금지된 매매 코드나 Redis 패턴이 나타나면 실패하는 정적 검증을 추가한다.

## 결과
- Phase 0는 시장 데이터나 외부 서비스 없이 검증할 수 있다.
- 이후 Phase는 Bit-Mania 런타임 리스크를 상속하지 않고 명확한 경계 위에서 진행할 수 있다.
